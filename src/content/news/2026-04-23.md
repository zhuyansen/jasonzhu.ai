### Better-skills：Agent Skill 版本管理工具正式开源

- **板块**：Skills 生态
- **来源**：Hacker News
- **链接**：https://github.com/ocherry341/better-skills

开发者自建的 Agent Skill 管理工具，借鉴 pnpm 思路实现 Skill 集中存储、多 Profile 共享、版本回滚和即时切换。解决了手动管理 Agent Skill 混乱、跨项目复用困难的痛点，对重度使用 Claude Code 或多 Agent 工作流的开发者极具参考价值。

### ChatGPT Workspace Agents 发布，Codex 驱动团队级自动化

- **板块**：AI 工具动态
- **来源**：OpenAI Blog
- **链接**：https://openai.com/index/introducing-workspace-agents-in-chatgpt

### Cursor 估值 600 亿美元被 SpaceX 收购？内幕曝光

- **板块**：AI 工具动态
- **来源**：TechCrunch AI
- **链接**：https://techcrunch.com/2026/04/22/how-spacex-preempted-a-2b-fundraise-with-a-60b-buyout-offer/

### Kimi K2.6 声称超越 Opus 4.7，成本仅十分之一

- **板块**：AI 工具动态
- **来源**：X/@bindureddy
- **链接**：https://nitter.net/bindureddy/status/2046987327957020909#m

### Ribbi：一只「青蛙 AI」一周吸引 4 万用户申请

- **板块**：变现案例
- **来源**：36Kr
- **链接**：https://36kr.com/p/3778121523025154?f=rss
