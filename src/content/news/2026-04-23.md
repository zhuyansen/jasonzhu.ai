### claude-context：让 Claude Code 读懂你整个代码库

- **板块**：Skills 生态
- **来源**：GitHub Trending
- **链接**：https://github.com/zilliztech/claude-context

Zilliz 开源的 MCP 插件，通过语义代码搜索让 Claude Code 拥有整个代码库的上下文感知能力。对于大型项目开发者来说，这意味着 AI 编程助手不再是「局部盲人」，可以真正理解跨文件的依赖和逻辑关系。

### Ribbi：一个对话框+一只青蛙，一周斩获 4 万用户申请

- **板块**：变现案例
- **来源**：36Kr
- **链接**：https://36kr.com/p/3778121523025154?f=rss

### 论文：AI「科学家」会出结果，但不会做科学推理

- **板块**：AI 论文
- **来源**：HuggingFace Papers
- **链接**：https://huggingface.co/papers/2604.18805

研究者对 LLM 科学 Agent 做了 25000+ 次运行评测，发现它们能产出结论，但推理过程并不符合科学的自我纠错规范。对开发者的启示：别把 AI Agent 当黑盒科学家用，流程设计和结果验证机制必须由人来把控。

- **板块**：AI 论文
- **来源**：HuggingFace Papers
