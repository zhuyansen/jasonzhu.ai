### Claude Code Routines 正式上线，自动化编程工作流再升级

- **板块**：Skills 生态
- **来源**：Product Hunt AI
- **链接**：https://www.producthunt.com/products/claude

Claude Code Routines 允许开发者将常用的编程操作封装为可重复调用的自动化例程，大幅降低重复性开发工作量。这是 Claude Code 生态中继 MCP 之后又一重要工作流扩展，进一步强化其在 agentic 编程场景中的核心地位。

### OpenAI Agents SDK 重大更新，支持沙箱执行与长任务代理

- **板块**：AI 工具动态
- **来源**：OpenAI Blog
- **链接**：https://openai.com/index/the-next-evolution-of-the-agents-sdk

OpenAI 为 Agents SDK 新增原生沙箱执行环境与模型原生调度框架，使开发者可以安全构建能跨文件、跨工具运行的长生命周期 Agent。此次更新直接面向企业级场景，标志着 OpenAI 在 Agentic AI 基础设施层面的竞争全面提速。

- **来源**：Google AI Blog
- **链接**：https://blog.google/innovation-and-ai/models-and-research/gemini-models/gemini-3-1-flash-tts/

### Chrome 推出 Skills 功能，AI Prompt 一键变成可复用工具

- **板块**：Skills 生态
- **来源**：Google AI Blog
- **链接**：https://blog.google/products-and-platforms/products/chrome/skills-in-chrome/

Google Chrome 上线 Skills 功能，用户可将常用 AI Prompt 保存为一键触发的工作流工具，支持发现、保存和二次混编。这与 MCP 理念高度契合，将 AI 工作流的入口从开发者下沉到普通用户，值得独立开发者重点关注。
