我用 Manus 做了一个分析 X 博主推文数据的插件。

真正花在输入需求上的时间，不到 3 分钟。

这件事本身有个很明确的信号：以后"做工具"，几乎不再是门槛问题。

真正的门槛，变成了：你知不知道自己要分析什么数据。

插件能做的事情很简单：

- 扫描指定博主一定时间内的原创推文
- 分析推文数据表现
