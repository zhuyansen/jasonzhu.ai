今天给企业培训，分享了我的网站开发心得

## 1️⃣ 0-1

**方式一**：直接上需求，慢慢调

**方式二**：需求分析与PRD → 架构设计 → 项目规划 → 测试用例、开发与调试 → 测试与部署

**方式三**：找对标网站，使用AI Website Cloner Template skills

**方式四**：找对标网站，使用AI Website Cloner Template skills，再加上front-design skills 和 UI UX Pro Max skill

## 2️⃣ 1-100 迭代思路

**方式一**：/insight 根据项目给我提些建议

**方式二**：使用gstack skills，让其项目进行审查，看看能不能提供一些点子

**方式三**：superpower下的brainstorm的skill，头脑风暴一些想法

## 3️⃣ SEO诊断

用seo-audit看下，给到我们一些seo建议

## 4️⃣ Prompt 工程 & AEO

## 5️⃣ 代码质量红线

文件/函数长度、缩进深度、Code Smells，CLAUDE.md 新增 Code Quality Red Lines 章节

OpenAI Codex CLI 进行AI Code Review（发现了 sitemap 计数 bug、关键词匹配 bug），修复了 2 个真实 bug，纳入日常 review 流程
