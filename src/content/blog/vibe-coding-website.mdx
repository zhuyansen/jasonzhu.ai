## 从0到1的四种方式

在最近的企业培训中，我分享了用AI进行网站开发的心得。这里总结了四种从零开始的方式。

### 方式一：直接上需求，慢慢调

最简单直接的方式。把你的需求告诉AI，让它生成代码，然后逐步调整。适合简单的页面或原型。

### 方式二：标准软件工程流程

需求分析与PRD → 架构设计 → 项目规划 → 测试用例、开发与调试 → 测试与部署

这是最规范的方式，适合复杂项目。AI在每个环节都能发挥重要作用。

### 方式三：对标网站 + AI Cloner

找到一个你喜欢的对标网站，使用AI Website Cloner Template skills快速克隆其设计和结构，然后在此基础上修改。

### 方式四：组合技

在方式三的基础上，叠加front-design skills和UI UX Pro Max skill，获得更专业的设计效果。

## 从1到100的迭代

网站上线只是开始，持续迭代才是关键：

- 使用 `/insight` 让AI根据项目给出改进建议
- 使用 gstack skills 进行项目审查
- 用 brainstorm skill 头脑风暴新想法
- 使用 seo-audit 进行SEO诊断

持续优化，让网站越来越好。
